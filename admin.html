Подключение

Токен нужен чтобы панель могла записывать коды в твой репозиторий.
Создать: github.com/settings/tokens → Fine-grained token → выбери свой репо → Repository permissions → Contents: Read and write → Generate.
Токен НЕ попадает в код сайта — только в память твоего браузера.
